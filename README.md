Overview:
This project provides insights into company financial performance through an interactive dashboard. It consolidates raw financial data, calculates KPIs, and visualizes trends to support strategic decision-making.

Key Features:
Revenue and expense trend analysis,
Profit margin and variance calculations,
Regional and departmental performance comparison,
Dynamic, interactive charts and visuals,
Automated data refresh for up-to-date reporting

Tools & Technologies:
Power BI Desktop / Microsoft Excel,
DAX (Data Analysis Expressions) for KPI calculations,
Data Modeling & Transformation

Usage:
Open the .pbix or Excel file in Power BI Desktop / Excel,
Refresh the data to see the latest financial metrics,
Explore dashboards with filters, slicers, and drill-down options.
